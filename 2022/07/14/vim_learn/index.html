Vim常用快捷键

发表于2022-07-14|更新于2022-07-19|Linux
|字数总计:936|阅读时长:3分钟|阅读量:

刚接触vim是在计算机导论课上（寄倒），上学期上软件先导实践课又接触了一点点vim。由于我记性不太好，先记录一下常用快捷键吧。

简介

vi编辑器是所有Unix和Linux系统下标准的编辑器，相当于Windows的记事本一样，它工作在字符模式，由于不需要图形界面，所以效率高。vim是vi编辑器的升级版本，除了兼容vi的所有指令以外，还添加了许多重要的特性，例如支持正则搜索，语法高亮，对C语言的自动缩进等，有代码补全、编译及错误跳转等方便编程的功能。

vim 键盘图：

使用

命令行

Linux命令行输入vim [[/目录/]文件名]

  • 若目标文件不存在，则新建空文件并编辑。
  • 若目标文件已存在，则打开此文件并编辑。

例如：vim /tmp/a.txt

三种模式

vi/vim 共分为三种模式，分别是：

  • 命令模式（Command mode）
  • 输入模式（Insert mode）
  • 底线命令模式（Last line mode）

快捷键

一般模式进入或者退出编辑模式：

按键 功能
a 当前光标后插入
A 在光标所在行尾插入
i 当前光标前插入
I 在光标行首插入内容
o 在当前光标下方插入空行
O 在当前光标上方插入空行
r 取代光标所在的那一个字符一次
R 一直取代光标所在的文字，直到按下 Esc 为止
Esc 退出编辑模式，回到一般模式中

编辑模式：

按键 功能
[Ctrl] + [f] 屏幕向下移动一页，相当于 [Page Down]按键
[Ctrl] + [b] 屏幕向上移动一页，相当于 [Page Up] 按键
gg 跳转至首行
G 跳转至尾行
0 移动到该行行首
$ 移动到该行行尾
p 将已复制的数据粘贴在光标下一行
P 将已复制的数据粘贴在光标上一行
yy 复制当前光标所在行
nyy 复制从光标所在行开始的n行
y1G 复制游标所在行到首行
yG 复制游标所在行到尾行
y0 复制光标该行行首
y$ 复制光标该行行尾
dd 剪切当前光标所在行
ndd 剪切从光标所在行开始的n行
d1G 删除游标所在行到首行
dG 删除游标所在行到尾行
d0 删除光标到该行行首
d$ 删除光标到该行行尾
nd$ 删除从光标所在行开始的n行
dw 删至词尾
ndw 删除后n个词
x 向后删除一个字符
X 向前删除一个字符
u 撤销上一次修改，可以撤销多次
[Ctrl] + [r] 重做上一个动作
.小数点 重复上一个动作

末行模式：

命令 功能
:q 不保存退出
:q! 强制不保存退出
:w 将编辑的数据写入硬盘档案中保存
:wq 保存退出
:wq! 强制保存退出
/word 向光标之下寻找一个名称为 word 的字符串
?word 向光标之上寻找一个名称为 word 的字符串
:s/old/new/g 在当前行中查找到的所有字符串old替换为new
:n1,n2s/old/new/g 在n1-n2行中查找到的所有字符串old替换为new
:%s/old/new/g 在整个文件范围内查找替换

有些快捷键有规律，比如移动复制剪切，先整理再多练。

帮助

命令行输入：

  • vim --help查看帮助。
  • man vim查看帮助手册。
  • vimtutor 查看教程。
